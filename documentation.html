Documentation

User Guide

Welcome to the College Help Desk! Here’s how you can use our platform:

  • Login/Register: Click the Login or Register button to create an account or sign in.
  • Ask Questions: Use the chatbot to ask any college-related questions. Select a department for more specific answers.
  • Request Documents: Go to the Documents section to request or download certificates.
  • Raise Tickets: If your question isn’t answered, raise a ticket for support staff.
  • View Notifications: Check your profile for important updates and announcements.

Admin Guide

Admins can manage the help desk by:

  • Managing FAQs: Add, edit, or remove frequently asked questions.
  • Viewing Tickets: See and resolve open tickets from students.
  • User Statistics: View user data and activity.

Developer Documentation

Developers can contribute to the project by:

  • Setting up locally: Clone the repository and open index.html in a browser.
  • Tech Stack: HTML, CSS, JavaScript. (For real projects: backend and database.)
  • Deployment: Push to GitHub and deploy on Vercel, Netlify, or Firebase.

FAQ

  • How do I reset my password? Contact support for assistance.
  • How do I update my profile? Go to your profile page.
  • How do I contact support? Use the Contact section or raise a ticket.
